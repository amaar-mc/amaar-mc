## Signal

I am a developer focused on the intersection of **AI research**, **agentic tooling**, and **full-stack product engineering**. I like systems that are fast, legible, and actually useful: optimizer experiments, local-first code intelligence, coordination protocols for coding agents, and web products that ship.

Current research thread:

- **FastMuon-NS3/NS4:** testing whether Muon-style optimizers can preserve most of their sample-efficiency gain with fewer Newton-Schulz iterations.
- **Agent infrastructure:** codebase context, conflict-aware collaboration, and orchestration patterns for AI coding tools.
- **AI-native products:** interfaces where models are not a feature bolted on at the end, but part of the core system design.
